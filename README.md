# Companion
Дипломный проект на тему: "Программное обеспечение для мобильных устройств на платформе iOS для коммуникации между студентами и преподавателями".

## Technologies
+ UIKit
+ MVC
+ Firebase
  + Auth
  + Realtime Database
  + Storage
+ Grand Central Dispatch
+ UserDefaults
+ Swift Package Manager
